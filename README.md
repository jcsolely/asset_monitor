# asset_monitor
账户资产查询
# 联通资产监控系统
基于 Flask + Vue.js 3 的联通手机账户资产监控工具，支持实时查询账户余额、流量使用、5G速率、已订业务和兑换记录，并提供流量变化监控功能。

## 核心特性
SM4 国密加密存储

- 用户敏感数据（手机号、Token、Cookies）和缓存数据均采用 SM4 国密算法加密后存储到 SQLite 数据库
- 加密密钥通过环境变量 APP_SALT 配置，未设置时拒绝启动，确保密钥不泄露
- 数据读取时自动解密，业务层无感知
退出即销毁

- 用户退出登录时，自动清除该用户的所有数据，包括：用户凭证、会话信息、缓存数据、监控记录
- 数据库中不留存任何用户隐私信息
安全防护

- CSRF Token 与 Session 绑定，防止跨站请求伪造
- 接口请求频率限制，防止恶意刷接口
- 安全响应头（CSP、XSS Protection、HSTS 等）
- 手机号脱敏展示
## 功能介绍

| 功能 | 说明 |
|:-----|:-----|
| 账户信息 | 查询余额、套餐信息 |
| 流量监控 | 实时查看定向/通用/免费流量使用情况，支持监控变化记录 |
| 5G速率 | 查询当前5G网络速率信息 |
| 已订业务 | 查看已订购的所有业务列表 |
| 兑换记录 | 查询积分/礼品兑换历史记录 |

## 配置参数

| 参数 | 说明 | 默认值 |
|:-----|:-----|:------:|
| `APP_SALT` | 应用加密密钥（环境变量，必填） | 无 |
| `LOGIN_EXPIRE_DAYS` | 登录过期时间（天） | `30` |
| `CACHE_DURATION` | 接口缓存有效期（秒） | `30` |
| `REQUEST_LOCK_DURATION` | 请求去重锁有效期（秒） | `1` |

# 安装依赖
```bash
pip install flask requests
```

# 设置密钥并启动（开发模式）
```bash
export APP_SALT='你的密钥' && python app.py
```

# 生产环境部署（推荐）
```bash
export APP_SALT='你的密钥' && gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

访问 http://localhost:5000 ，输入联通 Token 即可使用。
